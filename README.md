# International Workers Day Wishes App

A professional, reusable web app for **International Workers Day Wishes (May 1)** that any company can use to send employee appreciation messages.

## Features

- Auto/Light/Dark mode with saved preference
- Multi-theme engine with premium UI
- Personalized wish generator (name, company, role, tone)
- Multi-language support (English, Hindi, Hinglish)
- Rotating quotes and animated slogan tags
- Confetti celebration and micro-interactions
- PNG export, share tools, and speech synthesis
- PWA support with install prompt and offline cache
- Local engagement counters
